## 四、遇到的问题及解决方案

### 问题1：Netlify 部署后样式和交互失效
**原因**：上传的文件结构不正确，CSS 和 JS 文件路径无法找到
**解决**：创建 deploy 文件夹，将所有文件放在同一层级，然后整体上传

### 问题2：Tailwind CSS 版本错误
**原因**：CDN 链接使用了 `/3` 而不是具体版本号
**解决**：将 `https://cdn.tailwindcss.com/3` 改为 `https://cdn.tailwindcss.com/3.4.1`

### 问题3：GitHub Pages 不支持自定义文件夹
**原因**：GitHub Pages 只支持根目录或 docs 文件夹
**解决**：改用 Netlify 部署，更简单直接
